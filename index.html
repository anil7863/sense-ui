Sense your code

SenseUI provides real-time AI feedback to help visually impaired developers refine their designs.

How to use:

  1. Open the webpage you want feedback on.
  2. To open SenseUI, press Ctrl+Shift+S (or Command+Shift+S on Mac).
  3. Ask a question on the chat or use the commands (/).
  4. To add context or customize the feedback, go to Settings
  5. To read more, go to Documentation

SenseUI is still in beta and AI feedback might be inaccurate. Please report any issues.
