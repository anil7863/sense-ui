# SenseUI's Setup Manual
## Downloading 

1. Go to the [Releases page](github.com/reginacas/sense-ui/releases)

2. Download the latest release 

4. Open Chrome and go to the [Extensions page](chrome://extensions) (this link can only be opened with Chrome)

5. Turn on "Developer mode" (toggle in the top right)

6. Click "Load unpacked"

7. Select the folder called "src"

8. SenseUI should now appear in your extensions

## Opening 

To open it, press <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> (or <kbd>Command</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> on Mac)

## Pinning 
1. Open Chrome's [Extensions page](chrome://extensions)

2. Find SenseUI in the list of your extensions

3. Select "details"

4. Toggle the "pin to toolbar" option
