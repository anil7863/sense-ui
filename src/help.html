How do I open the extension?

Using the keyboard shortcut

Press Ctrl + Shift + S (or Cmd + Shift + S on Mac) to open the SenseUI panel.

Pinning the extension to your toolbar

  1. Open Chrome's extension menu (located in the toolbar palette next to the address bar)
  2. You can also access Chrome's extensions by going to chrome extensions page(opens in new tab)
  3. Find SenseUI in the list of your extensions
  4. Select "details"
  5. Toggle the "pin to toolbar" option

Can I change the keyboard shortcut?

Yes! You can customize the keyboard shortcut by opening the browser's extension manager. Click the "Open shortcuts page" button below to access it directly.

Important notes:

  • If you choose a keyboard shortcut that conflicts with a browser native shortcut, your custom shortcut will override the browser's default.
  • Resetting SenseUI settings will not reset this keyboard shortcut. To change it, you must return to Chrome's shortcuts page.

What commands are available?

SenseUI provides quick commands for common tasks. Type / in the input field to access them.

Available commands

  • /describe: Generates a visual description of the page
  • /issues: Identifies design issues ranging from critical accessibility problems to mild visual inconsistencies
  • /refresh: Re-captures page data (HTML, CSS, and screenshot)
  • /clear: Clears the current chat session

Can I ask custom questions?

Yes! You can ask anything about your webpage by typing a custom prompt directly into the input field at the bottom of the SenseUI panel. Simply focus the field, type your question or request, and press Enter to submit.

How can I navigate the chat?

User and System responses are structured with semantic headings that you can navigate using your screen reader's heading navigation shortcuts.

What actions can I perform on messages?

Each AI response includes action buttons (functionality will be implemented in a later sprint):

  • Copy to Clipboard
  • Mark as Favorite. This flags important messages to selectively download when you export your chat history.

What settings are available?

You can access the Settings page by navigating to the settings link in the navigation menu.

How can I add more context?

You can provide extra information to help SenseUI give more relevant feedback in the "Add Context" section:

  • Custom Instructions: Add specific requirements, preferences, or constraints for your project (e.g., "Always prioritize accessibility", "Use Material Design principles", "Maintain mobile-first approach")
  • Project Guidelines: Upload brand guidelines, design systems documentation, or project requirements documents (PRDs) to ensure recommendations align with your project's standards

How to report issues?

If you encounter any issues, please report them on our GitHub repository's Issues page.
