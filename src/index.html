SenseUI provides real-time AI feedback to blind and low vision developers.

SenseUI is still in beta and AI feedback might be inaccurate. Please report any issues.

By default, SenseUI analyzes only what's currently visible in your browser window (viewport mode). If you need to analyze an entire page, you can change this in Settings under Screenshot Capture Mode.
