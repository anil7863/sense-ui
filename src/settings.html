Screenshot Capture Mode
Choose which part of the page to capture for analysis. Captures the entire page from top to bottom. Slower, uses more tokens, provides broader overview. Not ideal for infinite scroll pages or sites with heavy dynamic content.
Captures only what's visible on screen right now. Faster feedback, uses fewer tokens, provides more detailed analysis. Best for analyzing specific sections or working iteratively.

Quick Action Buttons
Show or hide the Describe and Issues quick action buttons in the chat interface.

When hidden, you can still use /describe and /issues commands by typing them in the input field.
