Screenshot Capture Mode
Choose which part of the page to capture for analysis.

Important: By default, SenseUI only captures what's currently visible in your browser window (viewport). If you need to analyze an entire page, switch to Full page mode below.


Captures only what's visible on screen right now. Faster feedback, uses fewer tokens, provides more detailed analysis. Best for analyzing specific sections or working iteratively.

Captures the entire page from top to bottom. Slower, uses more tokens, provides broader overview. Not ideal for infinite scroll pages or sites with heavy dynamic content.
