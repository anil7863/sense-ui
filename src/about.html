User Manual

Opening SenseUI

Keyboard Shortcut

Press Ctrl + Shift + S (or Cmd + Shift + S on Mac) to open the SenseUI panel.

You can customize this shortcut:

Pinning the extension

  1. Open Chrome's extension menu (located in the toolbar palette next to the address bar)
  2. You can also access Chrome's extensions by going to chrome extensions page(opens in new tab)
  3. Find SenseUI in the list of your extensions
  4. Select "details"
  5. Toggle the "pin to toolbar" option

Commands

SenseUI provides quick commands for common tasks. These can be accessed by typing / in the input field.

Available Commands

  • /describe: Generates a visual description of the page
  • /issues: Identifies design issues ranging from critical accessibility problems to mild visual inconsistencies
  • /refresh: Re-captures page data (HTML, CSS, and screenshot)
  • /clear: Clears the current chat session

Custom Prompts

Ask anything about your webpage by typing a custom prompt directly into the input field at the bottom of the SenseUI panel. Simply focus the field, type your question or request, and press Enter to submit.

Feedback Detail Level

Choose how much detail SenseUI provides in its responses:

Comprehensive

Detailed responses with explanations of design principles, code examples, and learning opportunities. Best for building visual design knowledge and understanding the "why" behind recommendations.

Balanced (Default)

Clear, specific recommendations with brief reasoning. Includes numerical values and actionable steps without extensive code examples. Ideal for most workflows.

Concise

Direct, actionable recommendations only. Minimal explanations, maximum efficiency. Best for quick verification or when you already understand the concepts.

Additional Context

Provide extra information to help SenseUI give more relevant feedback:

  • Custom Instructions: Add specific requirements, preferences, or constraints for your project (e.g., "Always prioritize accessibility", "Use Material Design principles", "Maintain mobile-first approach")
  • Project Guidelines: Upload brand guidelines, design systems documentation, or project requirements documents (PRDs) to ensure recommendations align with your project's standards

Chat History Settings

Control what gets included when you download your chat history:

  • Download Entire Chat: Saves all messages from the current session
  • Download Favorited Messages Only: Saves only messages you've marked as favorites

Downloading Chat History (non-functional at this Sprint)

The chat history feature allows you to save conversations for documentation, team collaboration, or future reference.

To download:

  1. On the chat, navigate to the "Download Chat History" button
  2. Press Enter
  3. Your chat will be saved as a plain text (.txt) file to your default downloads location
