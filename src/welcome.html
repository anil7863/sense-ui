Welcome to SenseUI! 👋

The AI-powered design feedback for blind and visually impaired developers

To open and close it, you can use the keyboard shortcut Ctrl + Shift + S (or Cmd + Shift + S on Mac).

Get Started in 3 Easy Steps:

  1. 1. Configure Your AI Provider

    Add your OpenAI or Google Gemini API key in the Settings page. Don't worry, your keys are stored securely and never leave your browser.

  2. 2. Create Your First Project

    Improve and get more tailored AI feedback by creating projects. Switch between projects using a dropdown in the chat interface. For quick, one-off analyses of random websites, you can skip creating a project.

  3. 3. Get AI Feedback

    Open SenseUI on any webpage to analyze the current page within the chat using the commands available or giving a custom prompt. If you change something in your page, simply refresh it and the AI will analyze the updated version.

Important note

SenseUI is still in beta, and AI feedback might not always be accurate. Keep the same caution you would use when evaluating any AI-generated content.
