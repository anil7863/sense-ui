## What went well?

- Extension opened reliably with keyboard shortcuts across platforms.
- NVDA and JAWS successfully read responses and navigated headings.
- Interface is reported to feel overall intuitive and accessible.
- Use of `<h2>` tags enabled intuitive heading navigation.
- About page was helpful to testers.

## Sprint 2 planning

Sprint 2 will focus on:

- LLM integration (core AI functionality)
- Structured, navigable feedback responses
- Settings panel with API key configuration

With the next version, you'll be able to:

- Analyze webpages and receive AI-generated accessibility and design feedback
- Configure your own API key to maintain privacy and control
- If you don’t have an API key, please look into getting a free one from Gemini API (Google offers a generous free tier at https://aistudio.google.com/apikey – no credit card required, 15 requests per minute)

Privacy note: Your API key stays completely private on your device. SenseUI stores it locally in your browser—it's never transmitted to me or any external server. You maintain full control over your own API credentials.

Expected Timeline: 1-2 weeks
